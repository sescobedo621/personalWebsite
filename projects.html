Projects

  • redCarpetClub
    • Explore the best movies of all time with the redCarpetClub. Users can see all the movies listed, as well as searching by genre and name. They can also discover who is in the film. Users can also add their favorite movies, delete their least favorite, and update any information.
    • The redCarpetClub was made with Java Spring MVC in the backend. HTML and CSS was used in the front end. The information comes from AFI's Top 100 List
  • SuperWiki
    • Go up...up...and away with SuperWiki. SuperWiki is a superhero Wikipedia site and fan forum. Users can add their favorite superheros and villains, update any incorrect information, and talk with fellow fans about their favorite superhero.
    • SuperWiki is made with Java Spring MVC, JPA, MYSQl for the database, with HTML and CSS in the front-end
  • companyRolodex
    • All of an employee's information in one spot... companyRolodex is a human resources' dream. Users can see all employees, get specific information, add an employee, update their information, and delete if needed
    • companyRolodex is made with Java Spring MVC, JDBC and MYSQL database. The front-end was created with HTML and CSS
  • getFit
    • Whether you need to shed a few pounds or keep track of your New Year's resolution, getFit will keep track of your daily exercise routine. Users can follow of all of their previous exercises. They can also see their average daily steps.
    • getFit was made with Java REST and MYSQL database for the backend. It uses JavaScript DOM, HTML, and CSS for the front-end.
  • Whack-A-Mole
    • Go back to your childhood, and take your frustrations out with Whack-A-Mole. Users have 30 seconds to hit as many moles as possible, but more join in the fun as the timer winds down. List your name and see if you can beat the high score.
    • Whack-A-Mole was made with Java REST and MYSQL database for the backend to keep track of scores. The game itself. The front-end was made with JavaScript, HTML and CSS.
